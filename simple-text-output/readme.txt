== Description ==
 
Simple Text Output is a plugin that displays a text on a front-end using the shortcode [simple_text_output].
To use this plugin:
 
 1. go to Settings > Simple Text Ouput
 2. Enter the text you want to be displayed in your shortcode
 3. Put the shortcode [simple_text_output] wherever you want on your website.
 
== Installation ==

You can install Simple Text Ouput like every other plugins
 
1. In your admin dashboard, go to Plugin > Add new
2. Search for "Simple Text Output"
3. Click on Install
4. Click on Activate
5. Done!
